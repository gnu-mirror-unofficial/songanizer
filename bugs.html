GNU Songanizer - Bugs

Report a Bug

If you think you have found a bug in GNU Songanizer, then you should send as complete a report as possible to <bug-songanizer@gnu.org>. Do keep in mind that sending a patch is much more helpful than the mere bug report.

Bug Database

To have a look at all the bugs uncovered till date, you might like to take a look at GNU Songanizer's bug database. Patches are invited for any existing bug.
